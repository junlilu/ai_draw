Generate images from given textual descriptions

(1) Generate image from text, by given model (in generate)

(2) Train the model, generating images from text. (in train)

(3) Tokenize text (as Byte-Pair Encoding BPE) for English or Japanese text (in token)

(4) The model of neural networks and machine learning (in model)

Note that format of train data is text-to-image. e.g., Caltech-UCSD Birds 200 (dataset of birds)
